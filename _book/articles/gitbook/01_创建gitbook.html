Gitbook创建

安装

$ npm install gitbook-cli -g

查看安装后的版本

# 注意大写的V
$ gitbook -V

# 首次会安装
$ gitbook --version

转换

安装ebook-convert

生成电子书 (epub, mobi, pdf) 时需要ebook-convert

App下载地址

下载完成后可创建符号连接到一个环节变量地址中，方便命令行调用

$ sudo ln -s /Applications/calibre.app/Contents/MacOS/ebook-convert /usr/local/bin

转pdf、epub、mobi

$ gitbook epub
$ gitbook pdf
$ gitbook mobi
  • 目前存在排版和乱码问题，后续解决
